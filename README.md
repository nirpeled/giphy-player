# Giphy Player

![version](https://img.shields.io/badge/version-1.2.0-green.svg)

Giphy Player is a web app that plays GIF images using Giphy's API

### Running Instructions

```
1. npm install
2. npm run start
3. Visit localhost:3000
```

### Tech Stack

* React
* Redux
* React Router
* Node.js with Express
* Webpack
* Hot Loader
* Babel (ES6 support)
* lodash
* FontAwesome
* SCSS
